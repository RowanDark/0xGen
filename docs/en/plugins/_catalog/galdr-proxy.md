# Galdr Proxy

Galdr Proxy is the interception layer for 0xgen. It terminates client HTTP/HTTPS sessions, applies rules-based modifications, and records a tamper-proof history that other plugins can consume.

## Capabilities

- `CAP_HTTP_ACTIVE`
- `CAP_HTTP_PASSIVE`
- `CAP_WS`
